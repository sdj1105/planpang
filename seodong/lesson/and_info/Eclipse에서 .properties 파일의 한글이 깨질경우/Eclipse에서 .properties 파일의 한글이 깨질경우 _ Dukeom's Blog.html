Eclipse에서 .properties 파일의 한글이 깨질경우

2016/04/16Eclipseeclipse, 한글, properties, unicodedukeom
 
 
 
 
 
 
i
 
Rate This


Eclipse에서 .properties 파일을 열었을때 아래와 같이 한글이 깨질 경우

1
2
3
fail.common.msg=\uc5d0\ub7ec\uac00 \ubc1c\uc0dd\ud588\uc2b5\ub2c8\ub2e4!
fail.common.sql=sql \uc5d0\ub7ec\uac00 \ubc1c\uc0dd\ud588\uc2b5\ub2c8\ub2e4! error code: {0}, error msg: {1}
info.nodata.msg=\ud574\ub2f9 \ub370\uc774\ud130\uac00 \uc5c6\uc2b5\ub2c8\ub2e4.

unicode를 지원하는 propedit를 설치해야 한다.

설치 방법은 다음과 같다.

  • Help – Eclipse Market Place 에서
  • find 입력창에 properties editor 라고 입력하고 go 버튼 클릭
  • unicode를 지원하는 것으로 나오는 Properties Editor 를 찾아 install 버튼을 클릭하고 설치한다.
  • 설치가 완료되면 이클립스를 재시작한다.
  • properties 파일을 마우스 우클릭하여 Open With – properties editor를 선택한다.
  • 아래와 같이 한글이 정상적으로 보이면 제대로 설치가 된것이다.
1
2
3
fail.common.msg=에러가 발생했습니다!
fail.common.sql=sql 에러가 발생했습니다! error code: {0}, error msg: {1}
info.nodata.msg=해당 데이터가 없습니다.
